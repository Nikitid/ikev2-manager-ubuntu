# Журнал изменений

Заметные изменения проекта перечислены в этом файле.

## Не выпущено

- Документация приведена к единой русско-английской структуре.
- MTProxy (C, сборка из исходников) заменён на
  [mtproto.zig](https://github.com/sleep3r/mtproto.zig) (Zig, предсобранный
  бинарник) Александра Калашникова (sleep3r), лицензия MIT. Новый прокси
  использует FakeTLS 1.3, анти-повтор и активное зондирование-маскировку.
  TLS-домен задаётся при установке и не меняется; для смены требуется
  переустановка. Устаревший C-MTProxy и его правила iptables очищаются
  автоматически при первой установке нового прокси.
